## ğŸ¯ How to Play

1. Open `index.html` in any modern web browser
2. Click any empty cell to place your mark (X or O)
3. Players alternate turns automatically
4. First player to get 3 in a row (horizontal, vertical, or diagonal) wins!
5. If all cells are filled with no winner, it's a draw
6. Use "Restart Game" to start a new round
7. Use "Reset Scores" to clear the scoreboard
